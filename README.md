# PalindromeCheckerApp
The objective of the PalindromeChecker App is to design and implement a console-based Java application that validates whether a given string is a palindrome under different conditions, while strengthening core programming fundamentals and data structure concepts.

 UC2: Print a Hardcoded Palindrome Result Project: Palindrome Checker Management System Use Case: UC2 – Hardcoded Palindrome Validation Purpose: Demonstrates basic palindrome checking logic Main Class: UseCase2PalindromeCheckerApp Entry Point: main(String[] args)
Input: Hardcoded string value 
Logic Used: Character comparison from both ends Output: Displays palindrome validation result Data Structures: None (basic string operations only)
